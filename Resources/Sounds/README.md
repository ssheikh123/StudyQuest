# StudyQuest Sounds

Place short `.wav` or `.mp3` sound effects in this folder using these exact base filenames:

- `buttonTap`
- `success`
- `error`
- `coin`
- `xp`
- `levelUp`
- `badge`
- `purchase`
- `questComplete`
- `reward`
- `whoosh`
- `pop`

`SoundManager` loads these through `AppSound` and handles missing files gracefully, so the app will still run before audio assets are added.
